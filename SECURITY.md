# Security Policy

## Supported Scope

Security reports are accepted for:

- `cognitrace-extension` source code
- build and packaging scripts used by the mainline extension

## How to Report

Please do not open public issues for security vulnerabilities.
Send reports to: replace-with-real-security@cognitrace.ai

Include:

- vulnerability description
- affected file(s) and component(s)
- reproduction steps
- potential impact

## Response Targets

- Initial acknowledgement: within 5 business days
- Triage decision: within 10 business days
- Fix timeline: depends on severity and exploitability

## Safe Harbor

We appreciate good-faith security research and will not pursue action against reports that:

- avoid privacy violations
- avoid service disruption
- provide enough detail for responsible remediation
