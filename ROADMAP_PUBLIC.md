# CogniTrace Public Roadmap (30-90 Days)

## 30-Day Targets

- Chrome Web Store MVP submission and first release
- Privacy/data-use disclosures published
- Public OSS repository with governance baseline
- Initial evidence case collection from real users

## 60-Day Targets

- Improve analyzer reliability from real-world evidence cases
- Expand test fixtures and regression coverage
- Add clearer in-product explanations for integrity scoring
- Publish first contributor-focused technical write-up

## 90-Day Targets

- Stabilize a repeatable proof package workflow (analysis + replay + export)
- Launch community contribution themes (`good first issue`)
- Prepare phase-2 integration direction for optional API workflows

## Success Metrics

- active weekly users
- evidence cases submitted and resolved
- false-positive support cases with successful outcomes
- test coverage growth tied to user evidence
