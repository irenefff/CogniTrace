# Chrome Web Store Data Use Disclosure (MVP)

Use this document to fill the Chrome Web Store "Privacy" form consistently.

## Data Types

- Personal communications: Not collected intentionally.
- Personal info: Not required for MVP.
- Web history: Not used as browsing history; current page URL/title are used only for session context while recording.
- User activity: Writing interaction metadata (timing, event type, paste length) when recording is enabled.

## Purpose of Use

- Core functionality (authorship process analysis)
- Product improvement via aggregate diagnostics (if explicitly enabled in future releases)

## Data Handling

- Sold to third parties: No
- Used for advertising: No
- Used for credit/loan/insurance: No
- Data encrypted in transit: Not applicable for fully local MVP workflows
- Data deletion request channel: support email listed in privacy policy

## User Controls

- Recording is off by default.
- User manually toggles recording ON/OFF.
- User can uninstall extension to remove stored local data.
