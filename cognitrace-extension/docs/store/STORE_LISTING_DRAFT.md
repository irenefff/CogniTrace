# Chrome Web Store Listing Draft

## Name

CogniTrace - Proof of Human Writing Process

## Short Description

Generate process-based evidence of human writing through local interaction analysis.

## Detailed Description

CogniTrace helps students, writers, and knowledge workers show how a text was produced.

Instead of relying only on final-output AI detectors, CogniTrace records writing process signals (when recording is enabled), such as typing rhythm and paste behavior, then provides:

- writing session analysis dashboard
- replay timeline of writing actions
- exportable process certificate (MVP format)

MVP note:

- local-first analysis flow
- supported surfaces: Google Docs and Word Online
- recording is opt-in and can be turned on/off anytime
